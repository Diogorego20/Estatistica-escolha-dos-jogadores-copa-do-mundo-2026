# Validação Visual da Prévia do Projeto

**Autor:** Diogo Rego - Estudante de Estatística UFPB

A imagem `img/dashboard_preview.png` foi revisada visualmente após a geração do ranking estatístico balanceado por posição. O arquivo apresenta boa resolução, contraste adequado, título claro, assinatura de autoria e menção explícita à fonte dos dados. A paleta por posição diferencia goleiros, defensores, meio-campistas e atacantes, tornando a comparação visual mais compreensível para o README e para a documentação.

A primeira versão da imagem concentrava o ranking majoritariamente em goleiros, pois o índice é normalizado por posição e os goleiros apresentaram maior escala absoluta no índice final. A versão final foi ajustada para exibir os cinco melhores jogadores por grupo posicional, oferecendo uma leitura mais equilibrada e adequada para apresentação profissional do projeto.
